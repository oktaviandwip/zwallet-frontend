## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/oktaviandwip/zwallet-frontend/
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. run prettier

```bash
npm run format
# or
yarn format
```

4. Run the app

```bash
npm run dev
# or
yarn dev
```

🌟 You are all set!
